# 🧶 Dai Trancas

> Catálogo digital e portfólio para exibição de serviços de tranças e penteados.

O **Dai Trancas** é uma página web desenvolvida para apresentar o trabalho da profissional Dai, permitindo que clientes visualizem modelos de tranças, consultem informações sobre o serviço e entrem em contato de forma rápida.

## 🚀 Tecnologias

Este projeto foi construído utilizando tecnologias fundamentais da web para garantir leveza e performance:

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização personalizada, layouts com Flexbox/Grid e animações.
* **Font Awesome**: Ícones para redes sociais e interface.
* **Google Fonts**: Tipografia otimizada para leitura.

## ✨ Funcionalidades

- [x] **Portfólio Visual**: Galeria de fotos destacando diferentes estilos (Box Braids, Nagô, etc).
- [x] **Design Responsivo**: O site se adapta perfeitamente a dispositivos móveis e desktops.
- [x] **Navegação por Âncoras**: Menu que direciona o usuário para seções específicas da mesma página.
- [x] **Botão de Ação (CTA)**: Links diretos para agendamento via WhatsApp ou redes sociais.
